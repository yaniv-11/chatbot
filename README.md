# Yann's ChatBot – AI-Powered Assistant

A conversational chatbot built with Python and Flask that interacts with users in a natural and intelligent manner. This project demonstrates how to integrate a basic NLP-driven chatbot into a web interface using Flask.

---

## 🚀 Features

- 💬 Real-time conversation with the bot  
- 🧠 Rule-based or ML/NLP-powered response system  
- 🌐 Flask backend with responsive HTML/CSS frontend  
- 🔄 Persistent conversation history (optional with DB)  
- 🧪 Easily extendable to integrate APIs or GPT-based models

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask  
- Frontend: HTML, CSS, JavaScript   
- **NLP Engine: Gemini API
